# 经典俄罗斯方块 - 纯前端增强版

一个功能丰富的纯前端俄罗斯方块游戏，单文件实现，无需后端依赖。

## 在线体验

[https://look666s.github.io/021/](https://look666s.github.io/021/)

## 功能列表

### 核心玩法
- 经典俄罗斯方块规则
- 7-bag 随机系统（保证方块分布均匀）
- Hold 暂存功能
- Ghost piece 投影提示
- Combo 连击系统
- Back-to-back Tetris 加分
- 三种难度（简单/普通/困难）
- 操作回放功能

### 增强功能
- **加载画面** - 旋转方块动画 + 进度条 + AI 生成封面图
- **过场动画** - READY? GO! 开始动画 / 游戏结束震动闪白 / 升级数字闪烁
- **成就系统** - 6 个成就（初出茅庐、消行达人、速度之王、Tetris大师、连击之王、高分玩家），解锁弹窗 + 音效
- **排行榜系统** - 昵称输入 / 按分数或时间排序 / 记录昵称、分数、消行、等级、难度、时间
- **游戏分享** - 格式化文本一键复制 / Canvas 截图保存
- **操作引导** - 首次进入显示按键说明，支持"不再提示"
- **个人简历** - 导航栏"关于我"按钮，嵌入显示 PDF 简历
- **顶部导航栏** - 游戏标题、关于我、音效开关、主题切换
- **暂停菜单** - 继续游戏、关于我、成就、结束游戏

### 视觉效果
- 3 套主题（经典紫蓝 / 霓虹绿 / 暖橙）
- 方块发光效果
- 消行粒子特效
- 消行闪烁动画
- Web Audio API 音效系统

### 移动端适配
- 触控按钮操作
- 滑动手势支持
- 响应式布局

## 操作说明

| 按键 | 功能 |
|------|------|
| ← → | 左右移动方块 |
| ↑ | 旋转方块 |
| ↓ | 加速下落 |
| 空格 | 直接落底（硬降） |
| C / Shift | 暂存方块（Hold） |
| P | 暂停 / 继续 |
| ESC | 结束游戏 |

## 技术栈

- 纯 HTML5 + CSS3 + JavaScript
- Canvas 2D 渲染
- Web Audio API 音效
- localStorage 数据持久化
- CSS 变量主题系统
- AI 生成图片（加载画面封面）
